## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/taliakusmirek/vault.web.git
   cd vault.web
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
